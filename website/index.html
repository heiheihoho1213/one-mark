欢迎使用 OneMark

专注写作的 Markdown 体验——即时渲染、本地同步、多主题皮肤。

const hello = "OneMark";

为长文写作而生

不是又一个在线笔记，而是真正落在硬盘上的写作工具。

所见即所得编辑

WYSIWYG 即时渲染，标题、列表、代码块、表格、图片在同屏完成。

本地文件夹同步

关联磁盘真实目录，新建、修改、删除直接写回硬盘。

完整 Markdown 能力

GFM 表格、语法高亮、Mermaid 图表、链接与图片块。

撤销 / 重做

独立 History 栈，最多 50 步撤销，不打断写作心流。

五套主题皮肤

经典杏白、曜石、松针、冰川、赛博，随场景切换。

另存为 & 导出

一键另存为新 .md；双击 .md 用 OneMark 打开，融入现有工作流。

本地优先的工作空间

  • 关联文件夹 — 选择本地目录，侧边栏即项目树。
  • 拖放打开 — 将 .md 拖入窗口，自动载入所在文件夹。
  • 在上级目录中打开 — 右键快速定位到系统文件管理器。
  • 冲突检测 — 磁盘被外部修改时提醒，可选保留本地或使用磁盘版本。
